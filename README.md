# Кейс 1 - CheatCoders
Веб‑приложение для бизнес‑аналитиков, позволяющее создавать сводные таблицы на больших объёмах данных (10 млн записей, 200 атрибутов) с использованием ИИ для рекомендаций и интерпретации данных.
Чтобы развернуть сервер, обратись к нам, мы предоставим вам свой сервер и устройство 🤗

##Стек технологий

- **База данных**: PostgreSQL 16

- **Фронтенд**: React + TypeScript
- **AI‑модуль**: Spring Boot (прокси) + Mistral (LLM)
